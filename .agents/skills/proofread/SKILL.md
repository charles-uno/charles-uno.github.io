# Proofread Skill

Check spelling and grammar in Markdown blog posts to be published at charles.uno

## Usage

```
/proofread
```

## Instructions

Fix ONLY:
- Spelling errors
- Grammar mistakes (subject-verb agreement, tense consistency)
- Punctuation errors
- Typos

## Output Format

Change in place. 

Also list the changes made so the author can review them.

## Plugin Reference

Do not modify:

- HTML comments: `<!-- ... -->`
- Liquid tags: `{% ... %}`
- Autocard tags: `[[ ... ]]`
